# SpaceGame (Babylon Edition)

Just load index.html into Chrome (or any browser that supports WebGL) and play!

## Controls

Basic WSAD controls:

- `w` to thrust forward.
- `s` to reverse thrust.
- `a` to rotate left.
- `d` to rotate right.

## About

This uses the [Babylon.js](http://www.babylonjs.com/) game engine thing.
